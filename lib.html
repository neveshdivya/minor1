LIBERARY MANAGEMENT SYSTEM
LIVE CODE

This C code implements a basic Library Management System. It includes functionalities for both administrators and students. The system allows administrators to perform tasks like adding and deleting books, viewing the number of books, issuing books, and resetting passwords. Students can log in to view the books they have issued. The code employs a simple console-based user interface and maintains an array of books with their titles, authors, and availability status. Please note that some functions like "See Number of Books" and "Reset Password" are marked as not implemented. This code serves as a starting point and can be extended and enhanced for a more comprehensive library management system.

Step 1: Global Variables and Function Prototypes

The code begins with the inclusion of necessary header files and the declaration of global variables. Function prototypes for adminLogin(), studentLogin(), and various admin and student functionalities are defined. Admin and student usernames and passwords are set as global variables.

Step 2: Main Function and Menu

The main() function sets up a menu-driven interface with options for Admin Login, Student Login, and Exit. The user's choice is captured, and a switch-case statement is used to call the corresponding login function or exit the program.

Step 3: Admin and Student Login Functions

adminLogin() and studentLogin() functions prompt the user for a username and password. If valid credentials are provided, the user is presented with a menu of options.

Step 4: Admin and Student Functionalities

Admin Menu: Add Book: Allows the admin to add a book with a title, author, and marks it as available. Delete Book: Prompts for the title of the book to be deleted and removes it from the array. See Number of Books: Not implemented in the provided code. Issue Book: Allows the admin to mark a book as issued to a student. Reset Password: Not implemented in the provided code. Logout: Returns to the main menu. Student Menu: See Issued Books: Displays the list of books that have been issued to the student. Logout: Returns to the main menu.

FINISH

This code provides a basic console-based Library Management System in C. It allows administrators to perform tasks like adding and deleting books, issuing books to students, and viewing issued books. Students can view the books they have borrowed. However, some functionalities like viewing the total number of books and resetting passwords are marked as not implemented and would require further development. The code can serve as a starting point for a more comprehensive library management system with additional features and a user-friendly interface.
